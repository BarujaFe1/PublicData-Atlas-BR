# PublicData Atlas BR

Scaffold inicial do produto público de ingestão, validação e visualização de dados abertos brasileiros.

## Domínio MVP (proposta)

**Educação** — indicadores municipais/estaduais a partir de 2–3 fontes abertas versionadas, com score de qualidade, mapa e relatório metodológico.
