## Capture steps (Chrome)

1. Open Live Demo (hard refresh after deploy).  
2. Width ~1280px desktop.  
3. PT language first; optional EN shot for bilingual claim.  
4. Hide bookmarks bar; no personal profile chrome.  
5. Export PNG; keep under ~1.5MB when possible.  
6. Update README gallery if filenames change.
